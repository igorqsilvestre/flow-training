Este é um projeto de um aplicativo para celular de treino tabata. 
Nele você pode colocar colocar uma série de tempos para executar o seu treino.
Podera salvar o treino ou editar e também pode fazer um treino não salvo.
Também tem a parte que verá o seu progresso vendo os dias que foram feitos o treino no calendário além de ver a quantidade na semana feito.

O figma do aplicativo é: https://www.figma.com/design/HZLzz4yZFQAQREY9KlRFZg/FlowTraining?node-id=1-2&t=4bgNYaVhj4TjOhSN-1

O projeto foi feito utilizando o react com o expo-go.
